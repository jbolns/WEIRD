About

WEIRD is brought to you by the single-person team from polyzentrik.com.

Like it? Great!

Don't like it? Not great. I was really hoping you did.

But you can't argue with results. WEIRD can produce incredibly fast websites, even from Word documents. Additionally, if you use it well, it can give you pages that are incredibly sustainable, which I think is very cool.

It will also become easier to set up and use over time.

The preferred contact method is any of the social media channels linked in the top bar. That said, if you absolute must, you can also contact us via email

Current status

WEIRD is currently in version Beta -7.

It works, but it hasn't been stress tested sufficiently, so it might probably fail in various different manners.

Next steps

Subject to change, the priorities in the coming future are:

  • Bug hunting and bug fixing
  • Support for additional document types
  • Integration with popular cloud providers
  • Better documentation.

It's open source, so feel free to lend a hand or financial support.
