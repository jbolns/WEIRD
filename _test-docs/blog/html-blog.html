Blog entry from an HTML document

This blog post was written in HTML. It exists as an .html file which gets included in the final website as part of WEIRD's build process.

Obviously, HTML gives you the most freedom when it comes to the ability to include things. If there's an HTML tag for it, you probably can use it.

Headings and subheadings work just fine

Unordered lists also work:

  • Item
  • Item
  • Item

You can thrown in ordered lists, too:

  1. Item 1
  2. Item 2
  3. Item 3

Words and segments can be formatted bold and italics, and all the standard HTML tags work: underlining, strikethrough, and sup and sub effects, just to name a few.

Links work just fine too. Here's a to our home page.

What about images?

Of course. It's HTML. Images work.

Simply use the standard approach you would use in HTML.
